# Gestión de Cursos - Panel Administrativo

## Descripción

Este proyecto corresponde al panel administrativo de un sistema de gestión de cursos desarrollado con Angular. Permite administrar la información de los cursos mediante operaciones básicas de mantenimiento (CRUD), facilitando el registro, consulta, edición y eliminación de cursos.

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS
- Node.js
- npm

## Funcionalidades implementadas

- Visualización del listado de cursos.
- Registro de nuevos cursos.
- Edición de información de cursos existentes.
- Eliminación de cursos.
- Visualización del detalle de un curso.
- Servicio para la gestión de cursos (`CourseService`).
- Modelo de datos `Course`.

## Instalación

1. Clonar el repositorio:

```bash
git clone <URL_DEL_REPOSITORIO>
```

2. Ingresar al proyecto:

```bash
cd admin-angular
```

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar la aplicación:

```bash
npm start
```

5. Abrir en el navegador:

```
http://localhost:4200
```
